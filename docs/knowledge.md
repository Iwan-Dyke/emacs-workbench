# Knowledge

The workbench includes Doom's Org stack for notes and knowledge work.

Current first-pass behavior is intentionally light:

- Org support is enabled through Doom
- org-roam is included for a private knowledge graph
- notebooks are expected to use Org and Emacs Jupyter

This area is not wired into a custom workflow yet. Keep personal note locations,
roam directories, and machine-specific paths in local overrides rather than in
tracked config.

Use the README for install and launch commands. Use ADRs under
`design_decisions/` for rationale and tradeoffs.
